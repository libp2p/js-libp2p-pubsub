Intro

Installable via npm install --save libp2p-pubsub, it can also be used directly in the browser.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save libp2p-pubsub

You can then require() libp2p-pubsub as normal:

const libp2PPubsub = require('libp2p-pubsub')

In the Browser

libp2p-pubsub should work in any ECMAScript 2018 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of libp2p-pubsub, including index.js and index.min.js, are included in the /dist folder. libp2p-pubsub can also be found on unpkg.com under

  • https://unpkg.com/libp2p-pubsub/dist/index.min.js
  • https://unpkg.com/libp2p-pubsub/dist/index.js
